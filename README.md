# PSI Test Results

PSI 시험 결과를 정리하고 공개하는 저장소입니다.

이 저장소는 GitHub Markdown을 중심으로 정리합니다.
웹페이지는 보조적으로 제공합니다.
[https://postech-psi.github.io/test-results/](https://postech-psi.github.io/test-results/)

## 새 시험 추가

1. `tests/YYYY-MM-DD/` 아래에 새 폴더를 추가합니다.
2. 결과 파일은 `tests/YYYY-MM-DD/files/` 아래에 넣습니다.
3. 필요하면 Markdown 페이지, HTML 페이지, `tests/index.json` 을 수동으로 수정합니다.
4. 이후 시험 페이지 정리는 Codex로 처리할 수 있습니다.
